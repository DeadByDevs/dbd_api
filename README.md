# deadbydayapi

## required library : 

python-dotenv, fastapi, uvicorn

## how to set up the environment :

- create api-env
    - python3 -m venv api-env
- activate virtual env:
    - api-env/bin/activate
- install in env
    - pip install fastapi && pip install "uvicorn[standard]"

## how to run the app : 

python3 main.py
